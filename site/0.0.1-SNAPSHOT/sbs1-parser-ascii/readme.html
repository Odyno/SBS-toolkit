Module 2 - SBS-1 BaseStation ascii format parser

This format has become a semi-standard way of sending ADS-B data. I use as specification the documentation of protocol at http://www.homepages.mcb.net. The SBS-1 ascii messages are in a simple comma-delimited format; I show below examples of all possible messages ( http://www.homepages.mcb.net):
